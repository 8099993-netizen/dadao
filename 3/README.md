# 大盗号商 - Dadao Software Center

大盗号商是一个专业数字软件平台官网，使用原生 HTML、CSS、JavaScript 构建，可直接部署到 GitHub Pages。

## 文件结构

- `index.html` 首页
- `style.css` 全站样式
- `script.js` 全站交互
- `about.html` 关于我们
- `contact.html` 联系我们
- `privacy.html` 隐私政策
- `terms.html` 服务条款
- `404.html` 错误页面
- `favicon.svg` SVG 图标
- `sitemap.xml` 站点地图
- `robots.txt` 搜索引擎规则

## 功能

- 响应式布局，支持电脑、平板、手机
- 深色科技风、蓝紫渐变、玻璃拟态
- SEO Meta、Open Graph、Twitter Card、Canonical
- FAQ 折叠效果
- 数字统计动画
- 滚动进度条
- 返回顶部按钮
- 固定导航栏
- 外部联系方式均为用户点击后新窗口打开

## 部署

将所有文件放在站点根目录，启用 GitHub Pages 即可访问。

如部署域名不是 `https://dadao-software-center.github.io/`，请同步替换各 HTML 文件中的 `canonical`、Open Graph URL，以及 `sitemap.xml` 和 `robots.txt` 中的网站地址。
